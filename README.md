## 소개

DY LMS는 덕영고등학교 학생들을 위한 최신 학습관리시스템입니다.  
직관적이고 편리한 UI와 강력한 학습 도구를 통해 학생과 교사 모두의 학습 경험을 향상시킵니다.

---

## 주요 기능 ✨

- **맞춤형 학습 콘텐츠 제공**  
- **과제 및 시험 일정 관리**  
- **실시간 성적 확인 및 분석**  
- **공지사항 및 커뮤니티 기능**
